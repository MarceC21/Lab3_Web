# TheCatAPI – Lab 3 

Este repositorio contiene pruebas de la API **TheCatAPI**, incluyendo exploración de endpoints, pruebas funcionales y ejemplos de uso con Postman y HTTPie.

---

## Contenido del repositorio

El repositorio incluye los siguientes documentos principales:

### Postman

Aqui se encuentran el Environment y las colecciones creadas 

### Informe de API

Describe el proceso de onboarding de la API, endpoints probados, configuraciones de entorno y resultados obtenidos.

Incluye:

- Resumen de la API
- Configuración del environment
- Pruebas Happy Path
- Pruebas de errores
- Solicitudes reales
- Evidencia de respuestas


---

### Comandos HTTPie

Contiene ejemplos de requests realizados con HTTPie equivalentes a los utilizados en Postman.

Incluye:

- Variables de entorno
- Requests GET
- Requests POST
- Ejemplos de autenticación
- Ejemplos de errores


---

## Herramientas utilizadas

- Postman
- HTTPie
- Markdown

---

## Notas

Para ejecutar los comandos HTTPie es necesario configurar una API Key válida.

La documentación completa se encuentra en los archivos `.md` del repositorio.
